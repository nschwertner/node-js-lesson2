NPM usage examples:
```
   npm install MODULENAME
   npm list
   npm init      -> generates package.json
   npm shrinkwrap    -> generates full depenencies tree manifest
   npm install
   npm run
   npm run test
   npm test
```

NVM usage examples:
```
   nvm ls-remote
   nvm install 6.3.1
   nvm ls
   nvm use 6.3.1
   nvm deactivate
```
